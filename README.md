# JATLAS (Jav Actress Tier Ledger & Asset System) 
 
 **你的私人女优天梯与数字资产风控大盘** 
 
 ## 💡 为什么需要 JATLAS？ (The "Why") 
 
 对于拥有近万部本地日本 AV 收藏的“仓鼠党”和“囤积癖”玩家而言，传统的“按物理硬盘+文件夹”分类管理模式必然走向系统熵增： 
 * **人类记忆的极限**：面对几百位演员，你无法精准记住谁被分配在了哪个层级、存放在哪块硬盘。记忆偏差导致同一个演员的影片散落在多个物理目录中。 
 * **信息滞后的灾难**：女优从“现役”转为“引退”往往有信息延迟，导致你用现役的规则继续囤积引退的资产，打乱原有的存储逻辑。 
 * **无意识超载**：只凭感觉下载，缺乏硬性的“水位线”约束，最终导致低优资产挤占高优资产的物理空间。 
 
 **JATLAS 的诞生，就是用“数据库与规则引擎”对“人肉记忆”进行降维打击。** 结合 Emby 的底层刮削与 JATLAS 的顶层风控，实现对万级影视资产的高效、实时、自动化调仓管理。

 ## 🎯 核心特性 (Core Features) 
 
 当前版本已实现核心风控逻辑，未来将深度接入自动化流转： 
 
 ### 1. 动态水位与视觉风控 (Dynamic Quota Risk Control) 
 摒弃静态记账，引入动态水位线。系统允许为每个分类设定“建议保存数量”，实时对冲当前的实际影片数，并通过红绿灯系统进行强视觉阻断： 
 * 🟢 **绿色 (Safe)**：影片数量在建议水位线内。资产健康。 
 * 🟡 **黄色 (Warning)**：超出建议水位线 10% 以内。触发边缘告警。 
 * 🔴 **红色 (Danger)**：严重超标。必须立即执行“断舍离”删片，或提升该演员的天梯层级。 
 
 ### 2. 天梯与级联字典 (Tier Ledger System) 
 将资产的主观价值转化为可自定义的层级（Tier）定义，并与演员的生命周期（现役/引退）深度绑定： 
 * 系统支持根据**“现役 (Active)”**或**“引退 (Retired)”**状态，分别建立两套完全独立的分类字典与配额规则。 
 * 通过表单的级联选择，实现资产状态与物理存储预期的精准匹配，拒绝一刀切的混乱管理。 
 
 ### 3. 高信噪比看板 (High Signal-to-Noise Dashboard) 
 采用极简的冷灰 (Zinc) 控制台风格，剥离一切干扰信息的 UI 元素。一键检索演员，秒级定位其应处的层级与当前溢出状态，为你的磁盘整理提供绝对理性的决策依据。

 ## 🛠 技术底座 (Tech Stack) 
 
 JATLAS 采用经典且极速的前后端分离架构，专为单机/局域网环境打造，零外部依赖，极低心智负担： 
 
 * **前端 (Frontend)**：Vue 3 (Composition API) + Vite + Tailwind CSS + shadcn-vue。 
 * **后端 (Backend)**：Python 3 + FastAPI。极简、高可读。 
 * **数据库 (Database)**：SQLite3。单文件运行，随用随走，无需配置繁重的 MySQL 或 Docker。

 ## 🗺 演进路线图 (Roadmap) 
 
 JATLAS 不是一个静态的 CRUD 表格，它最终将进化为 NAS 环境下的自动化资产管家与偏好分析中枢。 
 
 * **Phase 1: 核心风控基座 (Done)** 
     * 完成演员与分类字典的 CRUD。 
     * 跑通“动态水位线”与“红黄绿颜色预警”的视觉风控逻辑。 
 * **Phase 2: Emby 实时对账引擎 (Next)** 
     * 接入 Emby RESTful API。 
     * 废弃人工手动录入影片数量，由后端定时或一键抓取 Emby 中演员的 `TotalRecordCount`，实现数据大盘与物理硬盘的 100% 实时同步。 
 * **Phase 3: 物理存储与自动化编排 (Future)** 
     * **NAS 环境感知**：接入局域网 NAS 设备的 API，实时监控具体物理硬盘的剩余空间。 
     * **软硬解耦重构**：基于规则引擎，自动生成指向物理视频文件的“软链接 (Symlink)”。调整女优层级时，只变动逻辑层的软链接，不再产生海量的物理文件搬运 I/O。 
     * **自动淘汰策略**：针对严重溢出 (红灯) 且未手动提权的女优资产，提供一键清理冗余低分影片的建议列表。 
 * **Phase 4: 动态天梯与喜好变迁雷达 (Dynamic Leaderboard & Trend Perception)** 
     * **资产级天梯打榜**：构建独立的天梯排位视图。打破单纯的分类聚合，支持通过手动拖拽或 Elo 胜率算法，对核心演员进行精准的绝对顺位排名，直观呈现最高优先级资产。 
     * **喜好异动前瞻预警**：基于时间序列的资产增量分析。当某个处于低优先级的演员，其影片数量在短周期内出现“异常拉升斜率”时，系统自动打上【Rising Star / 偏好异动】标签，提前感知并预判你的审美倾向转移，辅助你进行分类的提权调仓。
